# Assigment Developer - Pt Smooets

## Before Start

`npm i / npm install`

## Prepare Database

1. Create MySql databases with the table name 'profile_assignment'

2. `cd server`

3. `npx sequelize-cli db:migrate`

4. `npx sequelize-cli db:seed:all`

## Run Code

`cd server`

`npm run dev`
